# SuperHeroes-Phrase-Generator
The SuperHeroe Phrase Generator is a simple personal project built to practice Math.random and arrays, some of the concepts I learned in my first semester of the Systems Analysis and Development course.
The page has a simple layout: the title sits at the top, the phrase appears in the middle, and the button is at the bottom.
The page uses pastel tones throughout, keeping the design simple, clean and direct.

Hope you have fun! XD
